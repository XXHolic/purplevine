# purplevine
一个用于数据监控的收集和展示的系统。
## v 1.0.0
### 功能概述
- [X] 框架搭建，基本配置。
- [x] 主要界面:左边主要导航，右边展示错误信息列表。
- [ ] JavaScript 捕捉错误
- [ ] 接收的数据能够正常的存入数据库中，能够从中读取相应数据。

### 数据库设计
数据库表：log

字段 | 数据类型 | 功能描述 | 主键 | 非空
--- | --- | --- | --- | ---
id | INT | 序号 | 是 | 是
time | DATETIME | 接收时时间 | 否 | 是
ip | CHAR | 接收的请求 IP | 否 | 否
message | LONGTEXT | 接收数据具体内容信息 | 否 | 否
